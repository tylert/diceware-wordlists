Diceware Wordlists
==================

Examples::

    $ keepassxc-cli diceware -w beale.wl.txt
    honda civic his pick gag digit gt

    $ keepassxc-cli diceware -w francais.wl.txt
    mg fart rifles voire faudra revit gariez
